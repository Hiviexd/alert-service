# alert-service

web interface for alerting me when absolutely needed. (also a warm-up app for next.js)

## running locally

-   `npm i`
-   create `.env` from `.env.example`
-   `npm run dev`

### prerequisites

-   a [ntfy.sh](https://ntfy.sh/) setup
-   osu! api v1 key (not needed if `VERIFY_OSU_USERS` is `false`)
